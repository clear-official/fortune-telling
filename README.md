# 四柱推命の日｜杏月先生 簡易鑑定サイト

スマートフォン対応の四柱推命簡易診断サイトです。  
杏月先生による本格鑑定への誘導を目的とした、シンプルな命式鑑定ツールです。

---

## ファイル構成

```
/
├── index.html    # メインHTML
├── style.css     # スタイルシート
├── suimei.js     # 四柱推命計算ロジック
└── README.md     # このファイル
```

---

## 機能

- ✅ スマートフォン対応（モバイルファースト設計）
- ✅ 生年月日・出生時刻から命式を自動計算
- ✅ 命式表（年柱・月柱・日柱・時柱）の表示
- ✅ 五行属性（木・火・土・金・水）の診断
- ✅ 本質テキスト5項目の表示
- ✅ 杏月先生への本格鑑定誘導ボタン
- ✅ 日本語のみ（中国語・韓国語なし）

## GitHub Pages でのデプロイ手順

1. このリポジトリを GitHub に Push する
2. リポジトリの `Settings` → `Pages` を開く
3. Source を `Deploy from a branch` に設定
4. Branch を `main`、フォルダを `/ (root)` に設定して Save
5. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される

## カスタマイズ方法

### 杏月先生のリンクURLを変更する

`suimei.js` の末尾近く、`setupCTA` 関数内のURLを変更してください：

```js
btn.href = `https://あなたのURL.com/ankitsu?${params.toString()}`;
```

診断結果（名前・五行属性）がURLパラメータとして自動で引き継がれます。

### 五行の本質テキストを変更する

`suimei.js` 内の `HONSHITSU_TEMPLATES` オブジェクトを編集してください。  
各五行（木・火・土・金・水）に5項目のテキストが設定されています。

### デザインカラーを変更する

`style.css` 冒頭の CSS変数（`:root`）を変更してください：

```css
:root {
  --gold: #e8b84b;      /* ゴールド系 */
  --deep-navy: #0a0e2a; /* 背景色 */
  ...
}
```

## 計算ロジックについて

- 天干・地支の計算は伝統的な四柱推命の干支紀年法に基づいています
- 月柱の計算は「節入り日」（二十四節気）を基準としています
- 日柱は1900年1月1日（甲子日）を基点とした通算日数で計算しています
- 時柱は出生時刻不明の場合は「不明」として表示します
- 著作権フリーの伝統的な干支体系・五行理論を独自実装しています
